# Sofia Dias — Portfolio (Static HTML)

Pure HTML / CSS / JS. No build step, no PHP.

## Run locally
Just open `index.html` in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8000
# or
npx serve .
```

## Files
- `index.html` — markup
- `style.css` — styles
- `data.js` — content (projects, skills, education, marquee)
- `script.js` — renders content + handles smooth scroll and contact form (opens mailto)
- `assets/portfolio/` — project images

## Deploy
Upload the folder to any static host: Netlify, Vercel, GitHub Pages, Cloudflare Pages, S3, or plain shared hosting.
